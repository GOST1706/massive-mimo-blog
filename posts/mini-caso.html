Mini Caso: Beam Sweeping y SNR

Simulación de barrido de haces y análisis de SNR vs número de antenas para un usuario peatonal.

Este mini caso ilustra cómo Massive MIMO mejora la señal de un usuario mediante beam sweeping y cómo la SNR y la eficiencia espectral crecen con el número de antenas.

Código MATLAB

Puedes descargar el script completo para ejecutar la simulación:

Descargar beam_sweeping.m 
%% Script de Simulación de Massive MIMO
clc; clear; close all;
Pt_dBm = 20;
Nt_array = [4,8,16,32,64,128,256];
ue_angle_deg = 30;
beam_sweep_angles_deg = -90:5:90;
% ... resto del código resumido para visualización

El script muestra el barrido de haces, la selección del mejor haz para el usuario y el cálculo de SNR y eficiencia espectral para distintos tamaños de arreglo. Esto permite comprender la ganancia de antena (array gain) y la eficiencia del beamforming.
